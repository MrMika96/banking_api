# banking_api

### Запуск проекта

Для запуска проекта необходимо наличие Docker на ПК!

Чтобы запустить проект, необходимо выполнить команду, 
находясь в корневой папке проекта:

```
docker compose up --build
```

После удачного запуска, станет доступна документация API по адресу:

http://localhost:8000/docs/#/

### Тестовые данные

Чтобы в системе появились тестовые данные, необходимо выполнить
следующую команду

```
docker compose run --rm web sh -c "python manage.py import_test_data"
```
